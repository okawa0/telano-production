## **概要**

本サイトは、**実案件をもとにした仮想案件**です。

デザインは既存デザインを用い、**HTML / CSS / JavaScript を用いたフロントエンド実装のみ**を行っています。

---

## **担当範囲**

- コーディング（HTML / CSS / JavaScript）
- レスポンシブ対応
- SNSシェアウィンドウ実装

※ **デザイン制作は担当外**とし、デザイン支給で制作しています。

---

## **案件詳細**

- 依頼内容：HTMLでのコーディング。CSSは一つにまとめること。出来上がったものをWPへお客様側で組み込む。
- 納期：1~2週間
- 予算：6万5千円
- 基本要件：余白の調整、SNSシェアウィンドウの立ち上げ、記事画像保存防止処理、記事画像無しパターン想定実装

## **制作の目的**

- デザインカンプを正確に再現するコーディング力の証明
- 実案件を想定したディレクトリ構成・命名・保守性の確認
- 静的サイトにおける実装力の可視化

---

## **使用技術**

- HTML5
- CSS3 / SCSS
    - BEM記法
- JavaScript（ES6）
    - ポップアップでのSNSシェア機能実装

---

## **実装時に意識したポイント**

- **再利用性・保守性を意識したクラス設計**
- レイアウト崩れを防ぐための余白・比率管理
- 画像の最適化によるパフォーマンス配慮
- 縦長と横長での閲覧で表示を切り分けるため、ブレイクポイントを900pxで指定
